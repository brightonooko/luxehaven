# Images Folder

This folder contains all images used on the LuxeHaven website.

## Structure

- `luxelogo.png` - Website favicon and logo
- `apartments/` - Apartment gallery images (see below)

## Image Guidelines

- Use high-quality images (recommended: 1920x1080 or higher)
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Use descriptive filenames
- Maintain consistent aspect ratios for better gallery display
